# personal-blog-writing-coach

苏格拉底式采访 + 叙事结构，帮助写出中文第一人称个人博客（2000–5000 字）。

## 安装

把本目录完整复制到：

- Cursor：`~/.cursor/skills/personal-blog-writing-coach/`
- Claude Code：`~/.claude/skills/personal-blog-writing-coach/`

或放到项目内的 `.cursor/skills/` / `.agents/skills/` / `.claude/skills/`。

## 使用

对话中输入 `/personal-blog-writing-coach`，或直接说「用个人博客写作教练帮我挖素材」。

完整说明见仓库根目录 [README.md](../README.md)。
