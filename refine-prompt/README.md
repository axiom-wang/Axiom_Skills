# refine-prompt

把模糊需求改写成可执行的 Agent 实现提示词（先检查仓库，需求不清时先提问）。

## 安装

把本目录完整复制到：

- Cursor：`~/.cursor/skills/refine-prompt/`
- Claude Code：`~/.claude/skills/refine-prompt/`

或放到项目内的 `.cursor/skills/` / `.agents/skills/` / `.claude/skills/`。

## 使用

对话中输入 `/refine-prompt`，并附上你的想法或草稿提示词。

该 skill 默认只输出优化后的 prompt，不直接改代码。

完整说明见仓库根目录 [README.md](../README.md)。
